## ðŸ”§ Fine-Tuning

If something isn't working:

### HOTA Not Casting?
- Check "Enable Spell" is ON
- Verify "Only Bash Buffed" is enabled
- Make sure you have 2 Bash stacks (watch console messages)
- Check fury level (need 35+ fury)

### Too Much/Too Little Bash?
- Bash automatically fills in as needed
- If using too much, you might have other skills enabled that shouldn't be

### Skills Casting at Wrong Times?
- Verify min enemies settings
- Check filter modes on shouts
- Ensure orbwalker mode is active (press your orb key)

### Not Enough Mobility?
- Enable Leap if you prefer it over Charge
- Adjust charge/leap targeting
